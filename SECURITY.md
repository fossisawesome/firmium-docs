# Security Policy

## Supported Versions

This repository contains the documentation website for Firmium. It is deployed as a static site and does not handle user data, authentication, or sensitive information.

## Reporting a Vulnerability

If you discover a security vulnerability in this site (e.g. an XSS issue in the build/rendering pipeline, or a dependency vulnerability), please report it by opening a private security advisory on GitHub ("Security" tab > "Report a vulnerability"), rather than filing a public issue.

We will acknowledge reports within a reasonable timeframe and work on a fix as soon as possible.

For vulnerabilities related to the Firmium application itself (Android/desktop apps), please report them in the main `firmium` repository instead.
