# Contributing to Firmium Docs

Thanks for your interest in improving the Firmium documentation site.

## Getting Started

```bash
npm install
npm run dev
```

The site is built with Vite and Svelte. Documentation content lives in `src/content/*.md`.

## Making Changes

- Content updates: edit or add Markdown files under `src/content/`.
- Site/UI changes: edit the Svelte components under `src/`.
- Run `npm run build` before submitting to make sure the site builds cleanly.

## Submitting Changes

1. Fork the repository and create a branch for your change.
2. Make your changes and verify the site builds (`npm run build`).
3. Open a pull request describing what changed and why.

## Style

- Keep documentation clear and concise.
- Match the existing tone and formatting of surrounding docs.

## Reporting Issues

Open an issue describing the problem, including steps to reproduce if applicable.
